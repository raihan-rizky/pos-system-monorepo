# Produk

Gunakan menu Produk untuk menambah, mengubah, mencari, dan memantau produk.

Hal yang biasanya diisi:
- nama produk
- SKU
- kategori
- harga jual
- harga modal
- stok
- minimum stok
- unit

Untuk tambah produk, buka menu Produk lalu gunakan tombol tambah produk. Pastikan SKU unik dan minimum stok terisi supaya sistem bisa menandai stok rendah.
